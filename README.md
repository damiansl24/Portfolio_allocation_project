# Portfolio Allocation

I want to optimize my portfolio for long term growth while diversifying away as much risk 
as possible. So, I am using this repository as a sandbox for the python scripts
that I am going to write in order to optimize the portfolio. 

## Reused codebase:

My load_stock_data, log_returns, volatility, variance, and beta (alongside some other functions) 
will be taken directly from my more long-term project, Exogenous Market Shocks. 

## What this repository implements:

Annualized mean daily returns,

Annualized mean volatility,

correlation matrix (overlap sanity check)

Static covariance matrix (daily covariance * 252). 

3 year vs 10-year covariance comparison. 

For simplicity's sake, I will have two python files: reused functions, and new functions.
